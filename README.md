# MedicalCheckupSystem 体检管理系统
## 项目简介
基于 Java Swing + JDBC + MySQL 开发的桌面端体检管理实训系统，项目持续迭代开发中。

## 技术栈
- 开发语言：Java
- 界面框架：Java Swing
- 数据库：MySQL
- 持久层：原生 JDBC
- 开发工具：IntelliJ IDEA、Navicat

## 已完成功能
1. 用户登录验证
2. 体检套餐基础管理
3. 在线新增体检预约
4. 预约记录列表查询
5. 预约信息编辑、删除、刷新

## 待开发功能
1. 医生信息管理模块
2. 体检报告录入与查询
3. 表单输入格式校验优化
4. 管理员/普通用户权限区分

## 运行说明
1. 在 Navicat 执行项目对应 SQL，创建数据表
2. 修改 `utils/DBUtil.java` 内的数据库账号、密码、地址
3. IDEA 导入项目，运行登录窗口启动程序

## 补充说明
本项目为 Java 课程练手项目，适合初学者学习 Swing 桌面开发与 JDBC 数据库操作。
